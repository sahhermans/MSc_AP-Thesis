data_extracter		Script used to process the h5py output
					files of the coupled simulation. Output
					of this script are files that are used 
					by the rest of the scripts in this folder.
X.py				Script to process train and test data into 
					usable data loaders for the FCN.
s_zxy.py			Script to process train and test data into 
					usable data loaders for the convolutional 
					networks.
s_H2O_zxy.py		Script to process train and test data into 
					usable data loaders for the convolutional 
					networks, now incorporating water into 
					the input data.
